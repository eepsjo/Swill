# 啞缽

青石板剛潑過晨露，陶三的攤子已支棱起來。粗陶碗盞在草席上列成方陣，釉色肥厚的浮著鯉魚蓮葉，鑲銀邊的跳著鹿紋，最不濟的素碗也透著米漿色暖光。唯獨角落蹲著隻闊口圓缽，泥胎灰敗如積年灶灰，沿口爬著道蜈蚣似的裂痕。

「叮——當——」綢衫客的指甲彈過青瓷碗沿，清音濺得滿街脆響。陶三堆笑奉上錦盒：「前朝官窯的餘韻哩！」唯有那灰缽無人問津。頑童拿石子擲它，「噗」一聲悶響，似拳頭捶進棉絮。

張生便是這時撞進市集的。他肩頭還沾著祖宅的梁塵——今晨簽了田契，三十畝水田換得銀票一疊，懷裏像揣了塊燒紅的鐵。當夜他便買下城東三進大院，紫檀拔步床填滿內室，多寶格列著鈞窯瓶、翡翠山子，連茅廁都擺了尊鎏金貔貅。新婦過門那日，十六抬嫁妝塞滿回廊，新娘子腕間九重金釧叮咚，吵得池魚都躍出水面。

可三更梆子響時，張生總在拔步床上睜著眼。風聲穿過九曲回廊，在他骨頭縫裏嗚嗚地嘯。他疑心是屋子太空，又納兩房美妾。胭脂香浸透錦帳時，那風聲竟鑽進牙關，涼颼颼地往喉頭爬。

秋雨來得猝不及防。張生從當鋪出來，懷裏剛塞進一疊字畫銀票，忽見陶三在簷下舉著灰缽。缽口斜切雨幕，竟框住一弧驚心動魄的晴藍，似天神掰開烏雲漏下的眼。陶三手腕突翻，缽口猛扣進積水。張生幾乎驚呼——水中碎裂的雲影竟在缽底悠悠聚攏，旋如白蓮。

「這缽……」張生喃喃自語。陶三用袖口抹去缽沿泥漿：「廢窯裏掘的土，燒時柴濕了，盛水漏得慌。」裂痕在他指下蜿蜒，像道陳年的疤。

張生鬼使神差接過缽。雨水正順著裂痕滲入陶胎，涼意蛇一般纏上掌心。他學陶三舉缽向天，卻只兜住半截淋禿的柳枝。

當鋪門檻漸被張家人踏凹。紫檀家具抬走時，多寶格的空洞像豁牙的嘴；美妾抱著妝奁哭別，金釧聲碎了一地。最後一箱書離院時，張生坐在光溜溜的青磚地上，灰缽在懷裏溫吞吞地發燙。穿堂風撞進缽口，在腹內嗡嗡低旋，竟似深山古寺的晨鐘在陶土裏還魂。老僕拄著掃帚嗤笑：「老爺留個漏缽當棺材？」

張生不語，只將缽倒扣於地，湊耳去聽。風在缽腔裏打著旋嗚咽，陡然「叮」一聲清響，似冰棱墜入玉盤——可分明是旱天，院裏連露水也無。張生笑出淚來。他忽然明白暴雨那日，陶三扣缽時聚攏的碎雲是什麼——這滿院填不盡 風聲雨聲，原來早在這裂痕裏築了巢。

冬至夜，雪粒子敲得陶缽叮咚作響。月光爬上缽沿裂縫時，雪塵在豁口凝成冰晶，銀線般綴滿裂痕。破曉時分，陶缽已成了一盞冰琢的燈，幽藍的晨光在冰紋裏流轉。張生伸出凍僵的指頭輕觸冰棱，「咔」一聲脆響，某道深藏的陶裂終於綻開。雪光湧入裂縫的剎那，他聽見陶土深處傳來嬰兒初啼般的嗡鳴。

雪已停歇。空庭中央倒扣著灰陶缽，缽底積雪堆成小小的山巒。幾隻麻雀跳上缽肚啄食殘雪，爪痕在陶胎上劃出細密紋路。風掠過庭柱，陶缽忽然發出洞簫般的低吟，驚得雀群振翅飛入晴空。冰融的水痕順著新劃的紋路蜿蜒而下，像大地在飲一杯暖茶。
