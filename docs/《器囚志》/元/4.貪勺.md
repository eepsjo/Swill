# 貪勺

彼時青州府還未被喚作青州。城郊有一汪浩渺煙波，湖畔那座如今廢棄的橋頭斜插著一面褪色的酒旗，上書「貪齋」。店名狂悖，卻引得方圓百里的食客如蟻附羶。

那日，一名身著綢袍的官爺由隨從簇擁著踏入店門。這位大人平日裡吃慣了金齏玉膾，此番公幹路過，特意點名要嚐這「貪齋」中傳聞的絕味。小二垂著手，不多時，卻只端上一碟尋常不過的醬牛肉，配一壺土燒。

官爺眉頭一皺：「本官要的是聞名遐邇的奇味，你拿這等市井粗食糊弄，是嫌命長，還是欺我不識貨？」

後廚轉出一名枯瘦老叟，攏著袖子呵呵一笑：「大人息怒。小店只有這一樣菜，若是嫌淡，怕是大人心裡的火候還沒到。」

「放肆！」隨從正欲拔刀，官爺卻被那老叟深不見底的眼神勾住了火氣，冷笑道：「火候？本官走南闖北，什麼樣的珍馐沒見過？你這盤肉若說不出個子丑寅卯，今日便揭了你這招牌！」

老叟不慌不忙，只道：「這肉，要用那勺子吃，才算入了大人的口。」

「什麼勺子？」

「家傳的一點私器，不輕易見客。若是大人只是想填飽肚子，這盤肉就夠了；若是想填滿別的⋯⋯」老叟欲言又止。

官爺心頭那股被撩起的癢意愈發難耐，「拿出來。」

老叟又推脫了兩回，言說此物「燥性重」，恐傷了官家的貴體。官爺愈發不耐，重重擲出一錠銀子，老叟這才從懷中摸出一柄形制古怪的長勺。那勺柄極細，勺斗卻深，色如陳年臘肉，隱隱透著一股似香非香、似腥非腥的氣息。

官爺接過勺子，起初覺這長勺使起來極其笨拙，沉得壓手。他勉強舀起一片肉送入囊中，剛欲發作，神色陡然凝固。

第一盤肉下肚，他嚐到的竟非牛肉，而是三歲時在祖母桌下偷吃的一瓣糖漬紅橘，那甜味在喉間化開，竟讓他生出一股久違的純真。「再來！」官爺眼中精光暴漲。

第二盤肉端上時，味道竟化作了瓊林宴上的御賜鰣魚，鮮靈得彷彿魚尾還在喉間擺動，勾起了他初登仕途、意氣風發的快意。

接著是第三盤、第四盤⋯⋯官爺吃得愈發瘋狂，那勺子彷彿知曉他心底最幽微的渴求：南方的荔枝、北方的駝峰、甚至連他納妾席上的那口合儺酒，都隨著作響的勺斗翻湧而至。

「大人，」老叟在一旁低聲勸道，「肉已過五盤，再吃，怕是這勺子要反客為主了⋯⋯凡事有度，這滋味嘗到這兒，已是極限。」

「胡說！」官爺死死攥著那柄勺子，嘶聲道：「不夠！這天下所有的美味，本官已嚐遍了。可還有更進一步的？那種⋯⋯能讓人神魂皆醉，永世不忘的東西？」

老叟看著官爺那張因貪婪而扭曲的臉，幽幽一笑：「自然是有的。這世間最極致的滋味，不在山海，而在大人的『心頭好』裡。只是那道菜，非得大人您親自去『取』。」

老叟附在官爺耳畔，低聲耳語了幾句。官爺的瞳孔驟然一縮，隨即露出一種如痴如醉的、令人膽寒的笑容。

那夜，貪齋的後廚徹夜未熄，傳出陣陣奇異的切碎聲。

翌日清晨，橋頭霧氣彌漫。老叟在門口掛出新牌，名曰「貪肉」。那肉片薄如蟬翼，色澤紅白相間，紋理間竟隱隱有無數細小的銀絲，像極了某種精雕細琢的絲絡。

食客們一湧而上。有人吃出了萬貫家財的銅臭香，有人吃出了朱漆大門的厚重感，甚至有人吃出了權傾朝野的辛辣。每個人都吃得滿面紅光，彷彿吞下的不是肉，而是某人一生的野心。
