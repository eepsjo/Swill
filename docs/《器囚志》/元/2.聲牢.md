# 聲牢

刑場肅穆，唯風聲蕭索。典獄官踱步至那副業已鏽蝕的青銅編鐘前，其上綠斑斑駁，獸面紋模糊，音柱勾僂如老者脊骨。他命獄卒起出，親自熬煮魚膠，將鐘鈕重新澆鑄。

「縫進去。」典獄官指尖劃過最大那口鐘的裂口，聲音森然：「用浸油的牛筋線，從腳踝縫到下頜。」獄卒們面面相覷，盯著鐘腔內壁尖銳的銅刺。這刑罰遠比腰斬更窒人魂魄——人若成了鐘膽，便再發不出半點聲響。

受刑者乃琴師柳六，因調音時錯擰琵琶弦，致將軍在御前奏破，三百親兵盡數問斬。押解途中，柳六的耳朵總是緊貼囚車，似欲捕捉地底傳來的顫音。

魚膠未乾，柳六便被塞進銅鐘。牛筋線穿透皮肉的沉悶聲響，與編鐘殘片相撞的碎音混雜。當最後一針封住唇瓣，他忽然聽見幼年弄斷的第一根琴弦，在耳道深處「嘣」地炸開，奪去了所有聽覺。

青銅合攏的剎那，時間開始變質。起初只是黑暗，純粹而密實的黑，連眼球轉動的摩擦聲都清晰可辨。接著，有東西在耳蝸深處發芽——聲浪的鬼魂。將軍斷弦那夜的暴雨、刑場鬼頭刀破風的嘶鳴，甚至牛筋線穿透腮幫時纖維的撕裂，都在顱骨內壁反覆折射，無限膨脹。

第七日，鐘腔在收縮。青銅內壁的銅刺緩緩生長，刺尖抵住脊椎時，三百親兵的慘叫在他骨髓裡共振，那聲浪竟有了形狀：鋸齒狀的波峰刮擦著肋骨，圓鈍的波谷擠壓臟腑。他掙扎欲蜷縮，牛筋線卻將四肢釘成一個「大」字。

第一個雨季來臨，雨水在鐘頂匯成細流。水滴沿裂縫滲入，墜向額頭，撕裂空氣的嘶嘯被放大千倍，撞擊皮膚的震顫被拉長成永恆。那一滴滴水的爆炸，又在他腦海裡持續震蕩了三個月。

十年後的某個雪夜，銅刺已紮透肩胛骨。劇痛之中，他竟能分辨出雪片堆積在鐘頂的重量。每片雪的降落都像巨石砸落，三萬六千片雪便是三萬六千次山崩。當積雪壓彎老槐枝椏，「咔嚓」斷裂聲刺入太陽穴時，柳六忽然笑了。他認出那是十年前，琵琶弦斷裂的脆響。那時起，所有聲音都變了。

百年之間，王朝興衰更替，江山易主，刑場變成一片荒蕪。斷壁殘垣下，青銅長出新的銅綠，苔蘚從裂縫鑽進鐘腔，菌絲在柳六眼窩裡開花。某天，菌傘輕蹭過耳道，他聽見菌絲吸吮血液的滋滋聲，那聲音蔓延成網，網住將軍被賜死的白綾絞索、御前琵琶的裂帛，最終竟網成了母親紡車的嗡鳴。

「娘……」嘴唇翕動，牛筋線深深勒進牙床。

紡車聲驟然暴脹，青銅鐘體劇烈震顫，苔蘚孢子騰起綠霧。路過刑場舊址的行商，驚見雜草叢中凸起無數拳印，沙地裡滲出暗紅菌斑。

銅鐘重見天日，光線灌進鐘腔裂璺的剎那，柳六的耳膜似被無形利刃撕裂，三百年間積壓的聲浪終於找到出口，挾裹著這片土地所有的號令、所有鍘刀落槽的悶響、所有新鬼的咒罵，如海嘯般席捲他早已液化的腦髓。

柳六發現自己已然立在青銅鐘外。鐘體布滿蛛網般的裂紋，牛筋線仍縫著另一個「柳六」的軀殼。他伸手觸摸青銅，指尖竟穿過銅壁，像陽光穿過室內的灰塵。

鐘腔裡的柳六突然睜開眼，兩者隔著青銅對視。三百年的聲浪在目光交匯處坍縮成一個奇點，又無聲地爆裂。

「該調音了。」鐘外的柳六輕聲說道。

「當——」鏽蝕的青銅竟發出清越長吟，震得銅刺簌簌掉落，牛筋線根根崩斷，鐘腔內積年的血垢化作赤蝶紛飛。

當最後一根線斷裂，銅鐘轟然碎成綠雪，紛揚而下。附近的人們聞聲趕來，只見滿地銅渣閃著幽光。一隻聾了十年的老狗趴在銅渣上，耳朵緊貼地面，尾尖隨某種律動輕輕搖擺。

此後，刑場舊址一帶便傳說，子夜路過此地，便能聽見地下傳來琵琶聲；撥弦者不似人手，倒像有青銅器在自行震顫。月光最亮時，地裡還會浮出半透明的弦，繃在小草間。風過處，弦上滾落的不是音符，是細小的、閃著磷光的聲骸。
