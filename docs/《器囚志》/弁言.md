# 弁言

物者，心之囿也。<br>
人範金刳木，欲制六合之序。然器成則法立，法立則界生。<br>
界生而靈明自錮；<br>
猶造鎖者終困鎖內，築城者永隔城外。此造物之讖，眾生同縛。

痕者，存之契也。<br>
刃鑿所鏤，非止肌理，實魂靈拓於形骸之印。<br>
血沁斑駁，豈關妖異？乃存在抗辯之銘。<br>
器損而神彰，形破則魄顯。

骸者，道之筏也。<br>
器隳形銷，非歸寂滅：<br>
殘鐵化星潮之礫，燼塵作河漢之藻。<br>
蓋天地假碎器為觚，斟萬物齊同之醴；<br>
人造之牢潰於熵，未囚者遂游乎太虛。

今諸卷既啓，新獄方胎：<br>
寒匣欲囚春風，晶瓶思固流沙。<br>
然觀四野遺蛻，鏽鏈生碧，斷刃孕苔。乃知宇宙以破器為硯，血鏽為墨，書大化之誥。

諸君且觀：<br>
方鐐銬加身時，星潮已湧於指隙。<br>
此間無超脫之法，唯有殘骸鋪就的渡津；<br>
待君涉過自鑄之牢，看碎器如何孕出漫天塵光。
