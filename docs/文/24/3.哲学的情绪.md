# 哲学的情绪
> 本篇随笔的大部分内容来自笔者早年的研究，且笔者已经放弃该方向的研究，故内容可能缺乏客观中立性。抛砖引玉，请读者仔细斟酌，去其糟粕。

先从最近我摔倒的事说起。

当时被栏杆绊倒了，摔得很惨，直接跪下去还滑了半米。膝盖当场就破皮了，虽然现在看来伤势还好，但是若没有当时“伙计”帮忙送来碘酒和棉签及时消毒，不可能痊愈得这么快。

以上说话代表就我摔倒一事而言，虽然对“伙计”来说这是小事（他自己在微信跟我说的，虽然可能是客套话），毕竟从客观角度来说他只是拿起碘酒和棉签走路五六米进入我的寝室。但是这件事对我产生了很大的影响，原因有二，一是及时的帮助给了我很大的情感价值，二是我虽然不注重物质生活但我的灵魂毕竟被囚禁在肉体中，对于身体发肤，虽不着重美观，但也着重健康，特别是见血的伤势我向来是畏惧的。及时的消毒对我而言是很重要的。

于是就产生了物质层面和精神层面的出入。物质层面，如我刚才所说，“伙计”并没有付出很多，但在精神层面他给我提供了许多情感价值。类似于我给其他人搞电脑，比如前段时间的 Matlab，对我而言装一个 Matlab 是轻轻松松的，而对于其他人来说可能很难甚至无解（为什么有人愿意在网上花 20 块请一个未曾谋面的陌生人远程操控电脑？？？）。因此我为他们付出的物质代价和我给他们产生的情感价值就不对等了。

当然这是一个复杂的情况。变数有：我觉得简单的事，他们是否觉得简单（物质代价的多少）？我认为能给他们产生情感价值，他们是否买账？抛开 Matlab 的例子，因为阅读这篇随笔的人电脑水平可能很低（笑），就我摔倒而言，当是我的行动（走路）能力几乎没有，所以我是做不到找到碘酒/棉签这件事的，所以在这个相对简单的情况下，“伙计”做到了我做不到的事。不论如何，在物质代价和情感价值的考量中，社交关系诞生了一个记账标准，这笔帐是世上最难还的帐：人情。

产生人情的情感无外乎两种，感恩和内疚，前者的主观体验是其他人为你付出了更多物质代价/情感价值，因此你需要偿还他们；后者也是一样，只不过偿还变成了补偿，是一种消极的心态。因此它们本质是一样的，一个硬币的两面罢了。和其他人类情感不同，这类情感是由社会活动产生的。至于王阳明“良知说”，则是自然产生的“感恩”情绪，可能是生而为人感谢地球母亲，或者感谢生养你的父母的情感。注意家庭活动不算社会活动，因为没有社会的动物也可能产生家庭。虽然也有社会文化的影响，但是家庭活动更像是基因层面被设计好的东西。因此我们可以把感恩的起因分成自然道德和社会道德，前者更接近喜悦等不需要考虑社会活动的情感，因此这里的感恩之考虑社会道德产生的情绪。

让我们在深入一些。由热力学定律可知，宇宙的尽头是热寂，或者说封闭宇宙的结局是熵增。生命的意义在于更快地将太阳释放的能量转变成无法利用的热能，但是为了加速这一过程，生命在进化过程中是熵减的。自然道德产生的情绪，喜悦、愤怒、恐惧等，都不建立在熵减上，甚至愤怒的结果是破坏，而暴力破坏只能降低有序度，即熵增。社会道德产生的感恩，则是你需要为其他个体创造物质代价/情感价值的趋向，是创造性的情绪，是熵减的。这就是为什么人类进化出了社会，因为社会产生的情绪促使有序度上升。

社会的另一个产物是各种哲思，或者意识形态。意识形态也是具有情绪的。例如马克思主义哲学，它的根本情绪是愤怒。作为一种激进的主张，它煽动底层工人农民进行暴力革命，并尝试创造新秩序。但是我们知道工人农民大多是文化水平不高的人。民国初年军阀混战但除各别战役并没有非常大的冲突，因为这些军阀多是晚清遗老和贵族精英，文化水平较高，即使打仗也讲究先礼后兵。一战血腥荒谬，恰恰是欧各国国内民族主义情绪高涨所致。希特勒在一次公开演讲中说，纳粹主义（传统的日耳曼民族社会主义）和马克思主义（特指列宁主义）本质上是一样的。从哲学的情绪角度出发，它们确实都是愤怒的意识形态。

中世纪的宗教利用内疚情绪控制群众，现代大政府也用爱国主义等感恩情绪控制。战时，民族主义等愤怒的意识形态必然是高涨的，而和平时保守主义等较为温和的哲学会占主导。个体的情感是复杂的，但是群体的情感是单纯且可控的。控制好群体的情绪，可以快速组织一场战争，或者促进经济发展；而错误地使用群体情绪，比如冷战时美国入侵越南、苏联入侵阿富汗等，都是错误的使用群体情绪的例子。至于如何控制群体情绪，这是政治学家的工作，我们哲学家点到为止即可。
