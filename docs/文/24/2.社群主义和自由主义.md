# 社群主义和自由主义

人类制定法律的目的是实现“最多的人、最大的善”。启蒙运动中两种思想与它们提出的“社会契约论”都对如何实现这个目标给出了解决方案。

社群主义认为人性本恶，必须用强有力的大政府和法权来约束人民的自由意志。理想状态下，人民有自由信教的权利，但无神论是禁止的。此外，人民必须信奉“公民宗教”，即社群主义者规定的一系列政治正确。在社群主义社会，一切由投票决定，而投票又分“众意”的和“公意”的。前面提到的“公民宗教”教育的内容则是公意的，代表了法统所有人的“善”；而众意则是猥琐的、自私的。对于不认同“公民宗教”或者持有“众意”者，格杀勿论。

自由主义认为人性本善，不需要更不应该设立大政府。个人的权利必须得到尊重与保护，这是政府存在的唯一目的。在自由主义社会，一切由投票和立法决定，投票时必须尊重选民的个体意志。

从“社会契约论”的角度出发，社群主义是“平等”的契约，而自由主义则是“自由”的契约。前者的签约双方都是人民，签约内容就是“公民宗教”，因此不可能毁约：你不能在违反“公民宗教”的前提下替换掉签约的另一方，更何况对面和你一样，都是人民。后者的签约双方则是统治者与被统治者，签约的内容是法律，签约由投票生效。因此，若统治者违背了契约，被统治者有权替换它。

社群主义和自由主义都有其漏洞。既然社群主义认为人性本恶，那么为什么“恶”的人组成的大政府就拥有绝对的“善”呢？由谁规定“公民宗教”和“公意”？如果自由主义给予人们无限的个人权利，那么很容易产生一个社会丛林，很快会出现强势群体和弱势群体，而契约很容易被强势群体掌握。届时强势群体很有可能发展成为社群主义的“公民宗教”的制定者... 

社群主义对人类社会造成的伤害更加不可磨灭。法国大革命期间，无数旧贵族等被送上断头台，这就是社群主义格杀勿论的杰作。二战和冷战的战败方都是社群主义，它们的领导都是尝试设计社会的人。

上世纪七十年代，传统右派（也就是前面提到的自由主义）放弃了自由主义的称号，因为传统左派（也就是社群主义）在二战后也给自己戴上了“自由”的帽子。至此，传统左派成为“新政自由主义 (new-liberalism)”，传统右派成为“原初自由主义 (neo-liberalism)”，后改称“保守主义 (conservatism)”。

现代社会的各种政治组织不可能用简单的左右立场来分别。目前来看，各个政治组织已经开始融合两种自由主义的思想，甚至不乏“第三条道路”、“环保主义”等新兴意识形态。绝大多数组织都采用了拥有高法权的大政府，同时又保留了相当可观的个人权利。

融合两种不同的契约是不能长久的。绝对的平等就不可能存在任何自由，绝对的自由亦不可能存在任何平等。在平等契约的终点，人类可能会抛弃性别、肤色、高矮胖瘦，从一切基因的（物质的）区别到一切个性的（意识的）差异；甚至更进一步，有全人类融合成一个新个体，承载人类的全部意志（实际上是“公民宗教”），成为永恒的圣人甚至神。这就解决了谁决定“公意”，以及什么是“公意”的问题：因为个体自由意志不复存在。在自由契约的终点，人类可能会抛弃一切社会关系，如亲情、友情、爱情。他们会保留自己的个性；甚至更进一步，彻底地离散，达到“老死不相往来”的境界，每个人类个体再也无法通过自己的意志去感知其他人类个体的存在，达到永恒的离散。这就解决了弱势群体和强势群体的矛盾：因为大家都眼不见为净了。

从生物学的角度来看，生命从单细胞进化成多细胞，也有可能从单个体进化为组合体。蚂蚁、蜜蜂等实际上只是参与到蚁巢、蜂巢的一个小部分；而它们完全没有自我意识，或者说个性。因此为了避免人类融合，成为“公民宗教”的神，必须禁止任何人类基因改造相关的研究。

从热力学的角度来看，地球生命存在的意义在于更快的利用太阳产生的能量，使其加速转变为无用的热能。根据熵增原理，人类的最终归宿必然是毁灭，而在毁灭很有可能是熵更高的组合体形态。但这并不意味着我们就听天由命而选择殉教，因为生命的意义可不只是宇宙热寂的催化剂。这告诉我们应该更加珍惜自己的精神世界：只有在这里，我们才有无限可能，甚至是解决熵增的办法。

即使现在两大契约相处的很融洽，在未来的某一天冲突是不可避免的。我们活在高强力、高服从的社会，“公民宗教”并不由现任统治者制定，而是由文化劣根性决定。“公民宗教”的强力一天不消灭，人们的服从一天不改变，人类最终会沦为前人设计社会的陪葬品和祭品，为人类登神开拓一条血路。

前面提到一句老话，老死不相往来。西方国家的启蒙运动在后，封建统治在前，东方则相反。现有百家争鸣，才有封建王朝，因为生产力和经济的关系，不允许群众参与掌握了为数不多的生产力的统治阶级决定的政治活动，因此百家争鸣的无数思想，轻则沉没历史，重则沦为几千年“公民宗教”的教义，直到上世纪九十年代台湾野百合事件，才正式结束中华文化圈的“公民宗教”的绝对领导权。
