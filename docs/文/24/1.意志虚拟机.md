# 意志虚拟机
抛开小时候不谈，自从我的自我意识觉醒后，我就有一种强烈的感觉：所谓物质决定精神是错误的。第一个提出客观的人怎么证明别的自由意志真实存在（好吧，他们甚至不承认自由意志！）？人类所接受的一切讯息最终会化为主观体验，所谓客观不过是一套社会公理定义的、其他人的主观。但是正如科学无法描述人的自我意识的主观体验一样，我也无法证明意识可以脱离身体存在。虽然我在这套思想中否定了唯物主义，但是我不能否认科学意义上自我意识不过是神经元之间的电信号传递。

后来我从一个很简单的问题出发：假设一个人一天睡 8 小时，那睡眠时间就占了一天的 1/3。相比于睡眠，清醒时人的活动会更有意义吗？换言之，梦境和现实到底谁更重要。有的人睡得更久，也有人睡得更少。《夏日口袋》中，女主空门苍是这么描述睡眠的：睡眠时人在整理自己的记忆，记忆越多，整理的时间越长，人也就睡得越久。弗洛伊德《梦的解析》指出，人在睡眠时会把白天经历的不那么具体的事情在梦中复现。这些观点似乎都指向：梦是现实的延续。然而现实为什么不可以是梦境的延续？整理记忆时，多数人不能全盘接受，往往会给自己的记忆加工美化；在现实中不那么具体的事会走进梦中，梦里不那么具体的事也会走进现实。所谓既视感，有很多情况下是你做梦时碰巧遇到了类似的经历。

貌似梦境和现实的区别，只在于产生的记忆类型和对一个人主观体验的不同。现实产生的记忆往往是长期的，而梦境产生的记忆大多是短期，甚至不产生记忆。主观体验上，现实产生的劳累等直接作用于身体，是主观难以忽略的；梦境产生的各种感觉，可能醒来后就消失了。梦境就像是微缩的现实。

梦境主要以精神力为主。不要小看精神力。你可能觉得，我们得到的信息都是外界产生的，没有外界信息刺激，没有人跳出来告诉你什么是真实、科学、客观，人就容易误入歧途。实际上这是谬误。没读过现代科学的人可能可以琢磨出炼金术，没读过人文社科的人也有可能提出全新的哲学。我以前提出的“因果模型”和“瑞士东模型”就是这样的产物。精神力是无穷的，它脱离物质存在，不受物质法则的限制。然而我们人类灵魂确实被困在了身体里。所以我觉得人活着只要满足生命体征正常就足够了。如果能证明人死后自由意志依旧存在，肯定选择自刎。我不自刎的原因从来不是有人在乎，也不是我在乎别人。我害怕我失去思想的自由。

梦境和现实的问题后我又提出：人类的大脑是否存在更合理的运行方式。打个比方，人的大脑是计算机，我们用的道德标准、社会习惯是 Windows 系统。是否存在 Linux 系统，使得人类表现的和现在的社会行为完全不符。这是存在的，以前有过狼孩的例子。但是这明显不够好。有没有更好的系统呢？我们知道，GPU 不仅可以用来跑游戏，还可以建模、剪视频、挖矿、训练 ai。人脑处理不同功能的区域理论上有最优使用方式。《沙丘》中，门泰特就是被专门培养的人型计算机，他们的数理逻辑处理能力被大力开发。我们人脑是否存在这样的被开发的潜能？

目前为止人类仍处于高强力高服从的低级层次。这个理论来自强力-服从模型。正如我之前所说，我不是唯物主义者，对科学也并不完全认可，因此我的模型采用了基本的剃刀原理，把无关紧要的生产力因素抛弃了。人首先是精神的，不要物化概念意义的人。在这样的环境，人类社会的趋同化已经启动。按照进化论的说法，生命的基本单元细胞最终进化成多细胞生物，而自我意志的基本单位，人，是否会进化为多人的个体？《新世纪福音战士》中赤木律子说到：进化的终点是自我毁灭。我们当然无法要求组成自己身体的细胞去离散，因为我们是自私的人，不想抛弃自由思想的权利。我们能做的是阻止趋同化，实现人类的大离散，达到老子所说的：老死不相往来。

对于当前环境的一切社会活动和群体活动我都是极端排斥的。你可以说我懒，但我的真实感受是排斥。我讨厌集体住宿、班级活动、社区工作。但是我不得不怎么做：社会不养闲人。因此我研究出了一套保护机制，既解决了之前人脑系统的问题（至少是目前的最优解），又处理了梦境和现实的关系。

在讲保护机制之前，我要引用我和我的好友前几天吃饭时的对话。

我：听一个高中同学说，另一个高中同学人在上海，天天参加社团，自己搞乐队，参加各种学科竞赛。告诉我这些的同学说他很羡慕。

友：确实值得羡慕。

我：值得羡慕吗？我觉得他天天都在忙不属于自己的事情，没有提升自己。不如静下心读几面《佛经》。

友：读《佛经》就能提升自我吗？何况些活动怎么不能算是提升自我呢？

我：他只是提升了自己的社会活动能力，并没有提升自己灵魂的深度和层次。
> 插播一句：这里确实能提升自己的精神境界。但不多。

友：提升自我不能仅仅靠精神力量。就如锻铁术，如果一个人只看过理论，却没有实际操作过，那他很难锻出一柄好剑。

我：即使没有理论知识和实践基础，我认为正常人也可以通过之前拥有的各种生存经验去尝试锻铁。不过这固然很难锻造出好剑。不过，首先锻铁不是社会活动，其次它只是劳动行为，说白了就是干活，相比于精神与思想，就是下贱。

友：（笑）你这是脑子跟着屁股走了。

我：确实。回到刚刚的问题。我觉得他所做的不仅不是自己的事，还有可能导致自己进一步被趋同化。你所谓的自我提升不过是他逐步从拥有自我意志堕落到社会的螺丝罢了。

友：所以我们讨论问题三四年来，你的底层逻辑仍然没变？

我：首先我的底层不是逻辑。其次我的底层确实没变。

友：《佛经》也是逻辑。它是一套系统，自洽的系统。我觉得你的底层也是。如果不是的话你怎么说服自己？

我：我不需要说服我自己。底层从来不是用来说服自己的。至于“逻辑”，我以为你在说数理科学逻辑。

友：逻辑就是数理科学。

我：那我的底层不是逻辑。

友：无法理解。为什么你不需要说服自己？

我：（思考一段时间）举个例子吧：人是一台计算机，我的底层是一个数据库，而我在处理问题时，我会建立一个个虚拟机。这些虚拟机调用主机的数据 and 功能，但是虚拟机的数据不会进入主机，除非我手动放行。处理主机和虚拟机之间的正是你所说的逻辑。

友：我大概明白了。也就是说，你的是非是底层数据库，而在处理问题时只需要构建虚拟环境？

我：非也。我的数据库只储存数据和指针，不储存真值。负责处理一个数据的真值的是逻辑。不同的虚拟机我构建不同的价值观。和父母、和老师、和同学，我都用不同的虚拟机和他们交流。我的真实想法是没有立场。我只是把数据记录下来，知道它是什么，而不去在意它的对错。对和错本来就是可以篡改的，0 和 1 罢了。或者可以是分值，比如 100 分满分，我对一个事件的评价是 84 分。

云云。

对话中我提到的保护机制就是：自己只负责保留数据，不保留看法。当然我会有几个基本的原则，不过这不妨碍我在原则问题上变换立场。只有圣人可以言行合一，我们凡人两面三刀就可以了。这样做的优点是：真正的没有立场、大脑环境相对安全，不会被轻易改变。缺点也很明显：对我来说虚拟机切换速度挺慢。这也是为什么我不喜欢和人打交道。

和我相比我发现大多数人，包括我的朋友，要么是数据库里存了是非，要么是不用虚拟机而用相对轻量的沙盒；大多数甚至两个问题都有：他们的自我意志是不坚定的，立场的不稳定对他们来说是弊端。也很难指望他们想出全新的哲思。
