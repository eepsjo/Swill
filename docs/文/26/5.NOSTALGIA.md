# NOSTALGIA

我第一次看到璃月港的时候是下午；我指的是原神里的下午，现实是啥时间我已经忘了。当时我从望舒客栈出发，一路向南，想看看这片大陆上第二个城市的样子。绕过那个山角，落日的余晖泼洒在璃月港的街道上，和中式建筑的红润风格融为一体，映入眼帘。

不论游戏还是现实，夕阳总给我一种怀旧的感觉，下午发生的事我也总是记得更清楚；可能这就是我对怀旧感的刻板印象吧。小学生活已经很模糊了，但我总时不时梦见，五年级的自己在某个下午放学时刻，走在班级排队放学的队伍里，双眼无限聚焦望向远方，好像穿透了时光看到了现在的我。这些梦境有时是 pov，有时是第三人称（就像我远眺璃月港那样），还有时我站在以前的我的目光里，看着阳光从他背面射下，在地面上拉出长长的影子（因为身高原因我永远排在队伍的最后），他的眼神里带着某种不属于他那个年龄的无奈。

我的第一个 mc 存档是超平坦的创造档，当时我对这个游戏一无所知，在搭建的第一个房子的屋顶上，我放了艘船，却怎么也撸不掉。后来我索性换了个地方重新盖房子。当时的我不会疾跑，走到原来的房子已经离开渲染距离时，已是下午。超平坦的云层显得高而远，那颗我日后曾尝试飞上去的太阳正发出柔和的橘色光芒，北面升起的是一轮明月（mc 老版本 bug，太阳从北面升起），但我早就不记得月亮的盈亏。后来的其他存档和服务器不计其数，但我总是会想起那一个个夕阳，仿佛 mc 的时间被静止在下午，而我也被留在了原地。

我确实喜欢下午，特别是靠近傍晚，夕阳提起纱裙离去的那段时间。我觉得我人生中大多数美好的时光都在这个时间段。但记忆是不准确的；人的体验是会改变的。小学以为的下午，上了初中后才发现不过是第二节课下课；中学时的下午，到了大学可能还躺在宿舍床上，寻思是点外卖还是吃食堂。小学的我目光如炬，背后的队伍正随着其他人一起走出校门，很快操场就空了，教学楼里留下来参加兴趣班的教室发来些许嘈杂，学校西门的保安百无聊赖地彳亍着。他还站在那里看着我，那眼神中我品出了一丝意味：我是真实的吗？我是你吗？你还是我吗？

我和那位朋友最后一次见面是在一个下午。我和那座小镇最后一次道别是在一个下午。我犯下那件事是在一个下午。我喜欢上她是在一个下午。下午并不总是美好的回忆啊……唯一不变的是太阳，和晴空万里，或许飘着几朵云，但绝对不是阴天。那个小学一年级被学校推销去听的教辅软件讲座，我迷迷糊糊地坐到了第一排。主持人递下话筒：“小朋友，你的梦想是什么啊？”

“考上好大学，报效祖国，孝敬父母。”厕所在哪。我感觉全场听众都在盯着我，那个阶梯式的讲座厅好像一个坟墓，把我和我的尿意埋葬了。那个时候公交车好像是一块钱，但好像已经两块了。农夫山泉倒一直是两块的。

干燥的秋天，黑压压的乌云遍布了天空，这使得我无法判断这是上午还是下午。空气中瞬间充满了湿润的感觉，但静电并没有马上消失。“轰——”一声惊雷，我握紧了裤子口袋里的手机，那是我还怀疑手机会被雷劈的年代。我的心思随着雨水向天空飞奔而去，老师的粉笔在黑板上迅速地抹去了粉笔的痕迹，我松开了手机，所有人的耳膜和窗玻璃一起震动了一下，声波向着云层中同一个地方发射过去，雷声倒放的声音从大脑里滑落，卡在了时间的缝隙里。不多时，云层便散去了，原来是下午呵。

计程车师傅猛踩一脚油，我被惯性狠狠砸进座椅。车内有点热，橘红色的阳光洒在那台老旧的车载 cd 上，车子的音响正放着《红日》。小时候的我只觉得那旋律欢快，歌词一句没听懂；但是这有什么关系呢？那个纯粹的年代，纯粹的我，纯粹的快乐纯粹的世界。

我的公元前啊……那一个个日夜，独爱些橘红色的个把小时。世界在我们眼中，不都是从一个个尘歌壶，成长到地球这么大；有些区块被加载过，就再也回不去了。小时候总打喷嚏，把身边所有可能想我的人都考虑了个遍，却唯独没想到未来的自己正在记忆的匣子里不断地打捞。
