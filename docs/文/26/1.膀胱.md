# 膀胱

一阵强烈的尿意袭来，我不知道这是今天第几次了。因为开了小任务栏按钮现在只显示几时几分，不过看时间又是个通宵，所以这是今天的第一次。这冬天不冷也不暖和，不过不妨碍我只穿条裤衩子坐在电脑前写代码，这叫什么来着，心流，这个状态和死了差不多，人这个身体就超脱了物质的限制，对现实的感官就剥夺了，专注于工作本身。但是尿意这种通过什么激素影响什么下丘脑的好像作用点比较底层。我就不禁去想这世界到底是物质？还是说心流也有个限度精神需要放松。

和那个什么利尿激素的影响又矛盾的是现在巨他妈口渴。身体独居和精神独居感觉一样的，现在只能想那些不顺心的事，好像觉着快乐就是有罪，是吸毒后的幻想。烦心事越多就越不想动，天天点那个外卖吃，吃多了就口渴就喝水，又懒得去厕所，这边上议院不断地说，提出新需求，那边腚就往椅子上一长，手就往键盘上一粘，上议院的圣旨噼里啪啦地输入到终端，编译时世界静得可怕，电脑风扇的低鸣和电流滋滋声如雷贯耳。下议院快憋爆了，有的时候不如撸一发来的痛快。男性勃起的时候是没有尿意的。

冬天洗澡比夏天勤快，觉得冷了就去冲一冲热水，然后往被窝里一钻，睡不到几个小时又开始想那些烦心事，手办腿被磕了，脚本的逻辑感觉不对，这被子睡了一星期了是不是要去晒一下。洗澡的时候那个磨砂的玻璃把楼下棋牌室的灯模糊成渐变色，水雾上来后透进来，深蓝的暮色和 LED 的橘红洒在视网膜上，让我想起了 iOS 个位数版本的默认壁纸，然后想起了那个夏天玩的游戏，3G 时代移动端网游很少，单机多人很多。我就像那个游戏里的克隆一样，戳一下就跳一下，整个世界充满了电锯陷阱毒药管道，还在不断往后退，我一个人用四根手指玩四个角色，戳四下跳四下，然后就掉下来。落在后面的那个眼睛睁的大大的，你妈的你不救一下。多年后我也同样完四个手指的游戏，现在四个手指都平起平坐了。我开心不起来。

如果键盘有背光的话我大抵不会开灯吧。双拼对准度要求高，打对了爽打错了要倒回去老久，索性敲掉，洗澡时的热水浇在身上，电费还是燃气费在燃烧，人作为低熵体最后还是利用了熵最高的热能。秋天是短暂的，树叶从绿到金黄花了半年，从金黄到凋谢花了半天，风呼啸而过打在钢化玻璃上四散而去，又钻进混凝土森林的缝隙里，在冰冷的拥抱里加速到刺骨。门上的缝隙是你这辈子都不会抹灰的地方。Wi-Fi 信号好差，进后台看看。

我最终还是站起来走过了短短五米的距离。膀胱会爆的，我想。你们也是。我想，你们也是。
