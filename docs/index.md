# 遗忘的诺亚方舟

不知道多少人有我这样的习惯：想到一些思想、看了一部影片、和别人交流后，会在夜深人静的时候打开备忘录，记录自己的感受。 

有的时候，文章是经过精密构思的、反复打磨的，它们大多是议论文，简明地阐述我自己的观点；也有可能是散文，甚至小说。其他情况下，这些文章来自即兴发挥。 

我今天所想的，关乎一个人的自我认同，关乎记忆和遗忘、过去和当下。 

首先，一个人究竟来自他过去所有的经历，还是说他就*仅仅是*现在的“他”？这是很经典的问题。忒修斯之船——当所有木板被逐一替换，直至全船再无一丝旧痕，那它还是原来的船吗？人呢？构成我们身体的微小粒子，每一刻都在流动、代谢、更新，几年便几乎足以重塑一副全新躯壳。此刻写下这段文字的我，早已在物理层面与过去的我悄然断裂。 

记忆，大抵便是我们赖以维系“连续自我”那根脆弱的丝线了。然而遗忘何其汹涌，虚假也贯彻记忆始终。这就引来了今天的第二个问题：“真实”是值得我们去铭记的吗？或者说什么才是值得我们去留恋的？那些曾以为刻骨铭心的瞬间，那些曾灼烫灵魂的誓言，终究会在时光的冲刷下一点点剥蚀、模糊。

但只要现在能回想起来，与过去的自己产生一点点共鸣，便有他们被记住的理由。 

我总将那些即兴的感触匆匆投进这小小的方舟。它笨拙地承载起我生命之船上那些被时间巨浪冲刷、行将沉没的旧木板——那些“过去之我”的碎片。我固执地记录着，如同徒劳却虔诚的守墓人，深恐某一部分“我”就此湮灭于遗忘的深渊，连一声叹息都不曾留下。 

<ins>值得被铭记的，从来是自己。</ins> 

这诺亚方舟终日在电子海洋中漂流，它真能等到救赎的彼岸吗？那些被郑重其事存下的文字，如同被装入漂流瓶掷向茫茫大海的求救信。它们究竟是被未来某个时刻的我重新拾起，将过往的碎片拼凑起来，使那个消逝的“我”得以在理解中复活？抑或，它们将永远沉默地躺在数字的深处，成为无人认领的遗物，成为灵魂里一座座无主的孤坟？ 

我不得而知。我的手指仍不断地点击屏幕，固执地刻下又一行文字。这是我现在唯一能做的：在遗忘的洪流中，试图为那不断更迭、随时可能消散的“我”，留下一点存在的痕迹，纵使这轮廓终将淡去。
