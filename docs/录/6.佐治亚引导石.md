# 佐治亚引导石

保持人類五億以下與大自然-<br>
　永恆共存<br>
明智地引導生育增進健康<br>
　與變化<br>
以一種活的新語言來團結人類<br>
用沈著的理性來控制熱情－<br>
　信仰－傳統－及萬物<br>
用公正的法律及法庭來<br>
　保障人民與國家<br>
讓所有的國家自治在世界法庭<br>
　中解决外界的糾紛<br>
廢止瑣屑的法律及無用的官員<br>
讓私人的權力與對社會的義務<br>
　保持平衡<br>
珍視真－美－愛　尋求<br>
　與宇宙和諧<br>
不要做地球上的毒瘤<br>
　給大自然留點餘地<br>
