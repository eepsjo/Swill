# 俱往矣

这周天气灰蒙蒙的，但白天还是很明亮，路上会有一些小雨。走在路上，令我产生丝丝惆怅。这亮光与乌云就像我的未来一样，虽然模糊不清，但总归还是光明的。

然而，这小雨又将我带回到那中考完的下午，一切都是那么模糊又亲切。我记得，那天考完后，大家在一起鼓劲，不知所谓地齐声喊“周蔚”，路上有说有笑，然后各自回了家，似乎没有人在意各自人生中唯一的初中故事已经就此结束了。

我走出考场门后跟着爷爷，等姐姐走来，一起坐着公交车，聊了聊中考，回到了西湖矮矮的小家里。那天下的是梅雨，我穿着早已发霉的蓝白校服，身上汗和雨水混合着，和以前一样懒惰，没有去换掉。空气闷闷的，闻起来有一丝特别的味道，一股每年夏天都能闻到的味道。刚中考考完的我一直拖着这身汗，在空调房，漫无目的地划着手机。我不知为何当时我没有一种中考已经结束了的实感，但我确实很喜欢那天。我甚至没意识到，那可能是我最后一次待在那间待了九年的屋子里。

我记得，小学毕业的时候天也下着雨，lzx和lhf还在我家借待了一会。我和lhf一起看cc的视频，那时候我们的关系都很纯良，我不觉得他装，他也不觉得我唐，我们只是边看一个二逼卵子骂骂咧咧的说话边傻笑。参加完毕业典礼，我和lzx一起趴在冰凉的栏杆上，看着一个个班级的队伍被带出校园，也许这就是最后一次了。我仍清楚的记得，他略带愁绪地问我：“我们是不是再也见不到csr了？”

然而现在，我上完一周的学，撑着伞走出校大门。看着被雨水浸湿的路与灯光形成的美丽的光影效果，我又陷入了沉思。这一切都是那么清晰，雨丝打在我的脸上，令我感到凉爽，又让我感到深深的刺骨。就像我的内心一样，认清了许多现实，也发掘了真实的自己，总是感叹虚无，因为它清晰地存在着，并且我还一直坚信着保持这样思考的正确性，却在不知不觉间丢掉了那份青春的无知、酸涩，疯狂的动力与单纯的快乐。我反而有点讨厌这种清晰。也许只是因为我换了个眼镜吧，哈哈

我在地铁上，拿出了用lmh送我的绫波丽钥匙扣扣在一起的钥匙。不知不觉间与lmh看电影已经是十一月了，距上次和lsr出勤又过了一个月，今天又在学校看到了关于布置体育中考场地的牌子。在地铁上，我决定看几篇老师发的英语课外文章打发时间，那几篇文章都与时间有关，有对过去的回忆，有关于生死离别的故事，再一次令我陷入绝望和无力之中。我又想起我奶奶微信里的吴山中学老友群，也许不久之后，不经意间，我微信里也会出现那样的老友群吧。我真的很不希望那一天的到来，但我没有任何办法。前天音乐课上，放了“时光的河，入海流，终于我们分头走...”，说实话，我听着真的很感慨。

也许说到底，我一切的思考都来源于对时间流逝和死亡的恐惧。未来的我一定会想回到曾经的任何一个时刻，包括现在。我现在在干嘛呢？我躺在床上打着字。我不明白，我现在就挺想回到打出这些文字之前的。

我的初中经历也不全都很美好，有些时候大家对我也不太友善，我也犯过很多错，我为什么要怀念它呢？也许，就是因为它苦涩，才显得更为甜蜜。这看上去挺搞笑的，我究竟有多窝囊才说得出这些话？不过，我实在想和很多人说清楚一些事，道道歉，感谢一下他们，聊聊心里话。即使他们早就不在意了，大家都活在当下，活在未来，或许只有我活在过去了。

我原谅了zrf，原谅了ly，我想对ws表达同情，我想和不再假笑的yw谈谈以前的事，和敞开心扉的lmh好好聊聊人生，和不再表演的yyh交换观点，和耐得住想的ckw聊一道难题，和不再捂嘴的wj一起发自内心的笑，和放下脸面的st分享好玩的梗，和不动手的tyx讲讲猎奇段子，放下架子跟czx处好关系，不在hqk面前装傻，不再装逼，不再当一个小学生，不再开小团体，不再去猜测别人的心思，不再去冒犯他们或侮辱他们喜欢的事物......我以前常觉得我被耍是一件很令人委屈的事，但我自己做的似乎也不怎么好。我总是让人感觉到贱而不自知，甚至有些人起初对我很好，我却做出一副样子，说出一些话，于无形间伤害了他们。即使我是无意的，即使我后来意识到了，这一切都来不及了，只能将错就错。也许根本没有，在他们眼里我只是个招笑的小丑。不管怎么说，我诚心想和所有人都搞好关系，即使早就结束了。我放下了所有人对我的不好，也许那都是我自找的，我一点也不委屈。我想和放下心之壁的很多人说明白很多的话，我想让我的初中回忆饱和，我想让我的青春有一个happy ending。有时候，我挺后悔自己活成了 CoolStone09 的，我后悔我说过的很多话、做过的很多事。不过，即使一切变成了这样，即使这世界如此矛盾，即使时间还在不停的流逝，我还是微微一笑，“这样不也挺好的吗？”说实话，我想让这篇文章被初中同学看到，也许这样我所愿望的一切也许都能趋于实现了，但我找不到理由。

wrx又发了她回忆往事的长篇日记，她也想到了体育中考。我记得我体育中考的时候，还准备放松一些下次再努力的，结果一冲刺把三个都满分了。那天天气很好，也没让我感觉到很紧张的感觉。然而对她我也有很多想说的。要是我把这些发给她，也许她会深深感叹，也有可能她只会嘲笑我，即使我从来没见过她这样。我还记得，初二那年，我的笔袋被丢进女厕所，她主动提出帮我拿，我好像只是点点头而已。她还在和同学们出去玩的时候把我叫上，虽然她只是觉得我跟着他们比较能提供乐子。第一次那次，是我人生中第一次这样出去玩，后来基本上都是和lsr出勤了。说实话，虽然出勤我很开心，有人陪着我玩，但出勤总归是奔着打舞萌去的，我还从来没跟lsr逛过街什么的。然而她这么积极地让我融入团体，我却每次都不认真去对待，我还在那次跟bxy斗嘴，甚至我后来似乎还踩了她自己的雷，那次她还提出了我说的不好，然而我只是笑笑。说实话，我挺后悔的，我连这点都没能做到。也许她是班里最尊重我的人了，我却连仅剩的这部分都消耗掉了。她很完美，我不知道她维持这份完美是不是费劲的，至少她在人际关系上做得比我好得多，也许只有她真正有资格回忆幸福的往事。说到底，只是因为我太弱了而已。一个如此自以为是、情商低、外形臃肿、生活阴湿的人，配得上一个快乐的青春吗？配被人耐心地倾听吗？我必须承认，我搞砸了好多。对每一个初中同学来说，我的形象都多多少少挺糟糕的。

我可能确实太多愁善感了。Espejo 说，我这么放不下其实是因为这是我人生中在产生清晰的意识后经历的第一次分离。然而我看着现在放下过去的成年的她，也令我心酸。她追求平静；lzx也是这样，他想找到面对未来的方法和理由，然而同时他也在意着当下风景的美丽。我承认，他们这样是对的，但我不觉得他们是真正开心的。然而，我活在过去，难道就开心了吗？谁知道呢...

看着我在新同学群里发的没人回复的消息，也许我又无意冒犯了谁，踩了谁的雷，但我现在却不怎么会在意了，因为这样也挺好的。我喜欢EVA的原因就是这样，我们从来没有人能够真正的理解别人，也正因这样我们所有人都彼此独特。然而我以前总是感叹初中同学之间的心之壁太厚，同时他们又都是有心之人，或许有些人思考的比我多得多，就比如lmh。面对初中发生的那些不太好的事，我总说“这不是初中生该有的样子”，然而，初中生该是怎么样的呢？我好像并不明白。

反正，绝对的相互理解和自我意识是不共存的，我曾经就绝望于这种矛盾。我在这个世上是孤独的，我要建立一段关系就只能像豪猪一样持续调整，我不能完全的取到暖。我怕死，根本不是因为怕痛，根本不是因为怕死的“不体面”，而是我不想丢掉我的思考能力，和我宝贵而独特的记忆，同时我也害怕着那种绝对的虚无，所以我才想要永生啊。我不在意我是 CoolStone09，我甚至觉得这根本不重要。我明白了一件事，这世界上，唯一确定的幸福是我能够思考和感受，人生的意义亦如此，我根本不在意我这辈子能否有所成就，我从来没有因为我这辈子做不到什么而陷入虚无。

我还有很多想说的没有说，我还要很多想做的没有做，我依然很怕死，我依然感叹我在这世界上的孤独。但我又能做什么呢？我总想让别人完全理解我，哪怕只有一个人，然而语言总是有局限的。我以前很看不起那些写得很华丽的文章，我以为它们只是艺术品，没有感情，只是为了把那些忧郁哥看哭而已。现在看来，也许只有这种形式能让人理解我了，也许有人能透过这些文字，通过联想，获取一部分完整的我。我挺开心的，我现在真的挺开心的。我昨天晚上眼睛湿润地从地铁口中走出来，走到电动车旁边，发现爷爷放在袋子里的小纸条：“车保养过了，气打足了，速度也提上去了。雨天路滑，慢点骑，注意安全”。这世界仿佛瞬间明亮了。我骑在路上，抬头看看天，看看周围的世界，心旷神怡。到家又发现路灯装了起来，仿佛这个世界突然给了我一系列的幸福。然而现在又想想，未来会变成什么样呢？哎，还是活在当下吧，至少现在我是幸福的。

    嘿，我也写一篇文章送给你吧。

    这是一篇细腻又沉甸的文字。读着这些碎碎念，我能清晰地感受到那种独属于青春转折期的潮湿、敏锐与阵痛。
    
    这是我第一次收录朋友的文章，能在文中看到我说的话我很开心：“这是产生清晰的主体意识后，人生经历的第一次真正意义上的分离”。诚然，这种审视是痛苦的，它带来了巨大的无力感；但它绝对不“窝囊”，也不“阴湿”。相反，这是一个少年走向成熟时最勇敢的自我剖析。

    其实，人类记忆最奇妙的地方就在于，我们怀念的往往不是那个绝对完美的真空环境，而是那个环境里包含着的、再也无法复制的“可能性”。没有谁的青春是一座无瑕的雕像，大家都是在互相碰撞、误解、甚至伤害中摸索边界的豪猪。你怀念它，是因为那时候的每一次搞砸，后面都跟着大把可以挥霍的时间；而现在的你意识到，时间在收窄，代价在变大。这种苦涩，反向衬托了当年的单纯。

    那些想道却没能开口的歉意，一种“想让青春有一个 HE”的执念，是每个人在面对遗憾时的本能。但很多时候，生活的底色就是“将错就错”和“心之壁”。

    “这世界上，唯一确定的幸福是我能够思考和感受，人生的意义亦如此。”能在这个年纪深刻地体验到这一点，是你的幸运。你怕死，是因为你太热爱“存在”本身，太珍视自己这个独一无二的观测世界的视角了。语言确实有局限，它无法百分之百地复刻你的灵魂，但你看，你写下的这些华丽又真挚的文字，不就已经成功地让我、以及可能读到它的任何人，触碰到了你那颗敏感、潮湿却又闪闪发光的心了吗？

    文章的结尾，是一抹特别温柔的亮色。电动车袋子里留的小纸条，被保养好的车，加足的气，以及突然装起来的路灯。你看，这个世界虽然有“心之壁”，虽然时间在残酷地流逝，虽然未来模糊不清，但它总会在你被绝望围困的缝隙里，塞进一些极其具体、微小却坚实的幸福。

    不妨就听从你最后那句心灵的感悟——“还是活在当下吧，至少现在我是幸福的”。 过去已经“俱往矣”，未来的虚无还未发生，唯有此刻亮光，是真切属于 CoolStone09 的。
