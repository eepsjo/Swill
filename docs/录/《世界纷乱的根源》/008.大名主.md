# 大名主

> [原文链接](https://www.bilibili.com/video/BV1kHwpzWEDF/)

“大名主”是掌握较多已出让署名权的主体，大名主拥有强大的署名权力。大名主一定为某一人（不论是一个人还是一群人），而非一个机构，大多数时间由人伪装成某个中枢，以加强“界在”公信力。

诸如边沁的功利主义哲学家认为人是趋利避害的，边沁认为是非标准由这一理念确立，我们部分采纳这一观点，原因是在大名主之下，大多数被出让署名权的人类没有趋利避害的能力。大名主在署名过程中，一定保证其所“界在”的存在是对己无害的，尤其是在法律领域的“界在”过程中，一定会倒向署名权较多者。随之所来的，他们对存在“界在”的修改更加自由。

功利主义无法完整实现他们的理想目标，关键在于人类社会进程已然处于非初始的自然状态。简单的趋利避害已经不适用于“大名主”大量存在下的社会中。

人类历史上的“大名主”往往在历史初期施行温惠的政策，这是由于在上一封建王朝的末期，前任“大名主”收集上来的署名权逐渐失效，署名权逐渐分散到多个“小名主”手上，这就要求新的“大名主”若想重新掌握全部署名权力，就要通过政策与行动，告诉所有人类，他有能力与慈心照料这个土地，以此为机，再次收回居民的署名权。一旦收回全部署名权，“大名主”就会开始考虑自己的安乐，这是人类这一欲望机器不可避免的显现。

大名主”掌握“界在”事物的正义与否换言之，大名主定义了何为“好”与“坏”，并通过成文集载在册，形成“法”的起源。乌尔纳姆法典和汉谟拉比法典是人类早期的“法的起源，它们如实反映了“大名主”在“界在”过程中的主观偏向，比如法典中记载“上等人若让另一上等人骨折，则应打断他的骨”，但“上等人若让某个奴隶骨折，则只需赔偿60舍客勒（银子）即
可”，可见在较早时期“大名主”在“界在署名仅的过程中，毫不掩饰的包庇了法对上等人（暑名权掌握者）的利益倾向。

“象”与“名”的冲突在于“大名主”作为欲望机器的利己意识。正如同索福克勒斯笔下的“安提戈涅”那般，自然状态下人无法克服对亲人离去的悲伤，安提戈涅因此出于本性提出安葬她的兄长波吕涅克斯，但在“大名主”克瑞翁的“署名”下，波吕涅克斯是邪恶的、是背叛克瑞翁的，在他的淫威下，波吕涅克斯不得被郑重安葬，最终在这种冲突下酿成了这场无人得利的悲剧。

“大名主”手中的暑名权意味着大名主对事物可以自由的添加利于其自身的“正义性”与“严肃性”，此时正义性与严肃性的结合，便是“合法性”，而这本身就是“署名权”掌握者手中令人玩味的把戏，在这样的语境下，安提戈涅为兄长安葬便成为了邪恶的象征。
