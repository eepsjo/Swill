# 所有权

> [原文链接](https://www.bilibili.com/video/BV1WQPkzpEt5/)

所有权是“对一个物的署名权的完成”。狭义的所有权是对自己的财产享有占有、使用、收益和处分的权利,这仅围绕了你所能拥抱感触的物的所有权,这是世俗的所有权并非广延的所有权。所有权是“对一个象（征）的署名权”,是主体对一切“象”（征）的署名权。我们无法真实摸到“时间”,但是机构发布“宵禁令”却能通过对这段时间的署名,决定当地人口的户外活动时间。所以,这里所提到哲学化的署名权,是广延的,是作为署名权的结果。

“署名权——署名生效——所有权”,这是一个“象”从独立存在,到主体社会性附属存在的必要步骤。

署名权、所有权与“象”的本身天然存在一种微妙的矛盾,人类本能的占有“象”,但这种占有只能人类社会性的存在。

人类可以署名脚下的土地并生效成为己有,不令其他同类踏足,但猫、狗、蚂蚁、蟋蟀却但走无妨。换而言之,这种署名、占有与所有,是人类同族乐此不疲的把戏,一切宣称的署名与所有,从事实上只在同类间承认或存在,如同一座蚁穴,当蚁后在洞口宣告蚁群就此繁衍,只需人类的一次践踏与毁坏,便灰飞烟灭。

对于所有权的利益,永远是最大署名权人收获最高的利益,由于署名权让渡过程中,大量额外权利的流失,加之“虚假之名”的存在,导致权力急剧集中,主动让渡实际将走向被动让渡,当然,为了合法性,仍然会冠以“主动让渡”的堂皇之“名”。
