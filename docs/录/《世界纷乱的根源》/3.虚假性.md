# 虚假性

“名”有时是虚假的，但当大多数人承认这一荒谬署名权成立时，他便在“界在”中合法存在，直到人们收回署名权（如果此时还能收回的话）。

署名权的虚假性还体现在另一方面。洛克在论征服中断定政府起源不存在征服，而是绝对协商后的共同结合，但这一点忽视了“署名失真”的问题，这一问题体现在几个方面对契约论的驳斥：一是绝对的协商是无法达成的，这是象在面对多个主体表征不同而决定的；二是署名并非对等让渡，而是常常伴随额外让渡的；三是即便前两个方面勉强能够妥协，但祖辈署名权让渡后，因为血缘关系将这种署名权遗留给后代继续让渡，这本身和社会契约论缔造者的初衷背驰。

“署名权”本身确认了让渡在某些时刻的失真性，这是异于社会契约的根本，亦在警示人类主动撕开“虚假之名”。

名称，比如“家乐福”与“家家福”，他并没有在“界在”过程中根据自己实在的“象”而定名，而是特意“界在”为其他“象”上，以此获得另一存在在社会中的福利。

诸如以上所说，“名”的虚假性在于主体在“署名权”过程中的几个环节，一是署名让渡中的不自知的额外让渡；二是署名内容没有依赖存在，而是把两事物错误联系的署名起来；三是署名权利原本的自主收回权，遭到了无法回收的境地。
