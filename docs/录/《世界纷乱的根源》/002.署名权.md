# 署名权

> [原文链接](https://www.bilibili.com/video/BV19tAoz3EfQ/)

诚然，我们在认识“象”的过程中遇到了相当复杂的阻碍，但我们至少能够认识到“名”与“象”之间表达的合法性（在这里“法”的含义在于有能力而非有权力）。

“象”在大部分时间里，被人以“征”的形式感觉。“征”是所谓独立性或整体性的，他本身就是不同主体对事物的认识，即我们前文所述的“场测偏移”。类似于某些动物以复杂的气味辨别事物，语言和动作作为人类表达中最复杂而便捷的工具，被用来表征“象”。

有些人将萨特翻译成让-保罗·萨特或让-保尔·萨特亦或是沙特，但我们确信，不管是指哪种称呼，指的都是这位20世纪的法国哲学家。惠施曾提出“物方生方死”，我们在给予物“署名”之时，既可称他存续，亦可谓之消亡。通过这两个例子，我们可以明晰“署名权”不仅基于“象”复杂的特性，也基于主体对“象”的认识。

自然界中动物对于猎物的占有方式是多样的，有狮子或鬣狗的威慑性干预，又或是松鼠、貂之类的藏匿型占据，而“署名权”即从自然诞生。食物是人类社会第一次考虑占有的事物，这是远古时期人类捕猎的艰难而决定的。通常最具捕猎和采摘技巧的人更容易成为猎物的分配规则制定者，他们对猎物的“署名权”具有极高的自由意志，即此物“属于谁”、“谁能够享用”。当然，作为其他的人类（或动物）在察觉到有这样一顿朵颐饕餮时，他们也抛出了对物的“署名权”，但往往署名权的最终确认，属于自然状态下，机体更加强壮的主体。

部落的形成依赖于血缘的维系，这件事并不是人类独有的的，是自然界的。“署名权”尽管不能在自然状态下确保主体对物的占有，但署名权依然是绝对的权力根源，包括在占有猎物进食、占有领地休憩、占有配偶繁衍的基本自然功能，都源于“署名权”本身。从“象”“征”“名”的关系上来看，署名权本身宣告了主体对“象”的认识能否被承认。

进入氏族部落后，部落人的一部分“署名权”被贩卖给了部落首领。这种贩卖是个体为了生存利益的让渡。

现在，让我们重视“署名权”。一头野兽的捕获，作为部族的一员，最直接是为了获取食物来源，“署名权”是在争夺此“象”的食物部分。当部族首领，即最终的“署名权”掌握人分配完这一头猎物的血肉时，这场对象的署名却未完全结束。

一头鹿在被血肉啖尽后，他的皮毛显然可以制作成为衣服，鹿角则有药用，鹿头甚至又可以当作权力象征的挂件，这场“署名权”的背后仍有许多价值没有被深刻讨论，就草草的让渡了部族其他人的署名权力。

这一点就在于，不同的主体在看鹿这一个“象”的时候，他能看到多少方面的“征”，他是否能在食物价值的背后，看到药用价值、权力价值、威慑价值乃至更多价值。

但是很不幸的是，在部族首领披上鹿皮的时候，部的其他署名权出让者，已经不能再收回这一部分的权利。

在这个情形下，署名权出让者从理性上为了分配食物而向部族首领出让署名权以得到相对公平的分配（或仅仅是为了填饱肚子），但公正地说，接纳署名权出让者，在利用这个“名”（鹿肉）后就应当结束此次出让，并返还署名权，但由于上一章提到的“拥抱问题”那样，署名权的接纳者，用更加全面的角度审视一个名象后，发现了这场署名权背后的其他价值，他很难不会额外攫取这部分额外的收获。

霍布斯提出的社会契约和署名权的区别在于社会契约是纯粹经验的，社会契约签订的依据在于“桌子是桌子、椅子是椅子”，这就是我们前文提到的，认为名与征与象是对等的，是完全一致的，所以将社会契约理想化为一个绝对平等的约束。

但在“名象”的语境下，名是无法做到这样简单粗暴的抵达象的全部。在署名权的出让过程中，让渡者很大程度上会额外交出超出他们所认为的“名”之外的署名权力。

正如刚才的例子，部族的其他人类只是想交出分食肉的署名权，但却额外将分割皮草、鹿角的权力一并让渡了。这种署名权非出自自己本愿的额外让渡，我们称为“署名失实”。

到了封建社会，“署名失实”就日益严重了。农户出让庄稼的利润的“署名权”，另立为君主收税的名目。若农户收获庄稼卖出的利润是一个完整的“象”，他的“征”可以表达为“佃农给养家庭的收入”，也可以是“为得到君主庇护应当缴纳君主的税收”，农户出让署名权，放弃这部分给养家庭的收入，以寻求君主的庇护，但在“署名失实”的情形下，封建君主并没有完全按照他们原本的署名来进行，也没有实际庇护佃农，那么，这种署名的失真，就成为了佃农利益的长期损失。

当然，在封建社会中，哪怕农户认识到了出让署名权并不能换取“首领”的公正分配，但君主依然能通过强大的武力，逼迫农户出让署名权，即强制性的认可君主对这一“象”的署名，比如，出让农户脚下长期耕种的土地转而承认国家所有土地是君主的私有财产。

承认土地的所有权这件事情，是在为我们眼前的“象”（更多的是“征”）一个共识的“名”，署名这一形式，是人类将自然界归纳为人类“所有化”的过程。

“署名权”的让渡在于主体是自愿的让渡署名权力，接纳署名权的一方在重新署名的过程中确保执行署名出让方的意思，同时确保没有使用超出署名出让方许可的所出让权力。

简而言之，在自然状态下，土地的所有权是流动的，若要承认某一主体脚下的土地这一“存在”属于谁，那么，就需要出让“署名权”。
