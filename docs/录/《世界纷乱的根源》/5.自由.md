# 自由

> [原文链接](https://www.bilibili.com/video/BV1sCP4zkEFs/)

署名权是主体宣称的对“象”的占有，比如君主宣称占有所有土地及其附属的人口、建筑和资源，只需要土地上的人同意（不论主动或被动），并将署名权让渡给君主，君主便实际将其纳为所有。但事实上“名”只在人类中存在，君主并不直接拥有“象”，即便他无数次宣称该物的界在权。

“界在”从属于“存在“，“存在”来源于“或在”，对存在的“界在权”，是人人天生有之。

洛克曾论自由，自然状态，也是平等状态。在这种状态中，认为一切权力和管理都是相互的，无人拥有多于他人的权力，最为明显的是，同种和同等之物，生来就毫无差别地拥有大自然赐予的相同恩惠，利用相同便利条件。

但事实上，同类之间的机体差异不可磨灭，这忽视了人征服人的最基本原因——暴力。同时，何谓自由？自由不仅仅是处分自己的财产，也不是自由的生活，而是在这一些“自由的如何”之前先明晰何为自由，自由的根本就是按主体意愿的“署名权”。

当一个主体声明“我的身体是我的”，即界在自己的身体，赋予自我行动的支配权。当“我”作为主体，拥有完整署名权力的时候，“我”是自由的。对于“象”（或在）来说，主体有感受它的自由，进而形成“征”；对于“征” （存在）来说，主体有“名”（界在）的自由。这是一以贯通的（只为了描述这件事而如此叙述），以上这些让我们明白何为“自由”。

何为完全自由？即保留了完整的署名权力，界在“我”是“我”，土地属于“我”或“谁”，河流被谁占有，如果你仍然拥有完整的署名权力，你是自由的，署名权和自由是统一的。反之，当你让渡了署名权力，你便不再自由，换来的是更加安全、亦或是更加危险乃至战乱，这取决于你受让你署名权的“受让者”本身的取向。

奴役与自由是天然的天平两端，而中间的砝码，就是“署名权”，获得较多的署名权，则拥有更多的权力，得到了自由；失去署名权，则失去了权力，也失去了自由。

我们所说的权力，一切都来源于“署名权“，署名权是最根本的存在，例如人们将立法的界在出让给某一机构时，这个机构才能实施对这部分人适用的法律。当然，至于出让是出自本愿又或是强制，这是另外讨论的问题。署名权的出让有两种，一种是出于信任与认可对另一方管理者的主动出让；二是动用暴力或其他强制力夺取署名权的出让。

一个人、一个机构、又或是一个国家，在限定范围内拥有更多人口的署名权时，他们的权力与自由便更加庞大。

当一片土地，只有一个人的时候，这片土地是“免役”的，没有人遭受剥夺；当一片土地有一百人的时候，这片土地将会是“劳役”的，剥夺将以阵营形式出现；当一片土地有成千上万人时，这里就将出现“奴役”，因此人类从原始走向封建社会形成从上至下的秩序。

署名权的剥夺，导致每个主体失去宣称“何是何”的机会，在极端的情况下，甚至无法真实支配自己或承认“我是我”、“我属于我”。
