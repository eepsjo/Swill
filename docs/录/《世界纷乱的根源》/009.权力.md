# 权力

> [原文链接](https://www.bilibili.com/video/BV1Gxd8BVEGz/)

“或在“被感知为“存在”，“存在”被主体“界在”，群体性的认可或强制性“署名权”出让则诞生了“大名主”。

“大名主”的署名权全部来自于人这个主体，有多少人出让了全部署名权，那么大名主本身就对应掌握了多少权力。如果没有人口，大名主的署名权就会消失——当你站在四处无人的百丈山野，用尽全身力气咆哮呐喊，除了徘徊在山中的回音外，并无一人能为你做任何事，哪怕只是简单的为你斟水一杯。

因此，人是署名权的来源，只有足够庞大的人口，大名主才能掌握资源）被认可掌握），资源不是固有被某个主体真正掌握，而是被承认掌握，是被署名的。

若世上仅有一个人宣称一口湖泊属于他的私产，那是万万不可能的，他也并无法占据上万平米面积的湖泊，但若是有更多的人承认并去维系这一署名，那么这种人类之间建立的占有便成立了起来。

当你麾下有一百名工人，你雇佣他们为你挖掘钻石，并通过各种分销渠道将产品运出去到达消费者时，在这个环节中，工人如果没有将署名权出让给你，你便无法得到他们开采的哪怕一克拉钻石。工人这种从事实际开采，但出让了对手中钻石所有权的署名权行为，被撰写在了雇佣合同中，这就是署名权出让的法律化体现，通过这种方式，界定了劳动的目标与产品的归属。

诸如亚当斯密的哲学家认为劳工价值的矛盾源于资本所有者对工资、地租、利润的划分不均，但更加深远的是署名权出让与受让的问题。例如，矿主因获得多少署名权受让获得此地采矿权？署名权由谁给予？是否有署名权来分配利润？分配利润的署名权又是谁给予等等。因此，仅讨论分配环节不均的问题，仍然是浮于表面的，而没有触及更加深层的、在开采过程中一系列署名权的细节。

署名权是人类社会自诞生起天然的权力，人口繁盛的前提下，大量署名权被转移到各个大名主手中，他们逐渐掌握了更多署名权。

署名权在这里最简易的理解就是“界在人如何感受的或在的）即界在存在的）权力”。我们在前章提到过，对“象”的理解是愈发深入的，每一个象”是一个无限“征”的集合，那么，在大名主收集署名权的时候，就会出现一个情况，例如农民出让土地给富户后，富户倾向于占有土地不仅是开垦权，乃至流过此处的河流权、土地下的矿产权，土地上房屋的所有权等等，这就是署名权出让后的困境，农民仅仅以用来耕种土地的价格出让给了富户，但他却失去了这块土地的未来的所有可能性。但署名权的出让必然决定了额外权力的丧失——＂象＂的更多‘征”在被发现，但人们已经提前将对“象”的署名权出让了。

理想化的署名权是“我出让这片土地供你耕种，受让者也仅用来耕种而已”，但现实是，署名权的受让者贪婪的扩大权力，突破授权限制，攫取更多署名权获得后的收益。

署名权出让给“大名主”后，就意味着大名主对这一“象”或“征”署名权的解释内容，出让者都要尊重且执行。同时，一旦出让署名权后，这一权力将难以收回。
