# 正义性

> [原文链接](https://www.bilibili.com/video/BV1URcrzSE17/)

正义是美的延续，美是生命的延续。

这是被共识认可的，因为人被设计成趋避死亡、谋求生存的群体。能延续生命的，是自然的，便是美的，美德是看起来能帮助他人延长生命的行为，正义是美德的化身，这是从人的特性出发的署名共识。

诸如正义性、严肃性这一“名”已经被普遍作为共识认可，他们常常用来修饰新“名”，以达到这种“美名”的延伸。正义性是“名”对“象”赋予的产物，一切被称作正义的事件和行为都首先需要被“界在”何为“正义”。正义首先是不伤害自己的，没有人会称伤害自己性命的事是正义的，因此杀害生命通常被认为是邪恶的。在后来的部落和城邦构建中，正义又被逐渐赋予为维护特定人群或宗室家族的行为，例如“兄友弟恭”“忠不违君”被称为正义的美德，这是为了维护宗法和家族的稳定性而被增赋的正义性。

秉性、品格与美德，乃至“美”与“德”这些“名都是被赋予的产物，人被赋予掌握署名权。署名权随着人类社会流动性，在世界被塑造的过程中，诞生了越来越多的“名”。他们像标签一样被贴在了“征”的表面，将他们分类而别，最终在一定时期内形成了“好”与“坏”的区别，随着大名主的变换，这些“标签”也会随之更迭，“名”被重新赋予——他们本就不是真实的存在，更无所谓如何变换，只需要被大部分人所承认便可。
