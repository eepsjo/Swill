# 纷乱

> [原文链接](https://www.bilibili.com/video/BV1b6V86eEAx/)

从“名”走向“征”再走入“象”，是逐渐走向真实的道路。例如一个立方体，我们需要多个角度观察他的“征“才能知道他是一个立体的并有六个正方形平面构成。而每一次的肉身性的直接观察，就是这个事物的“征”，包含了对其一切的主体性认识，不论是感官上的还是意识上的。“象”是这一切的集合，是不断超越上一次对存在认知并接近于“象”本身的突破。“名”是对“征”乃至“象”的界定命名，其中有恰当的，也有虚假的，有相对客观的，也存在相当主观的。

“名”是纯粹主体性的、中心性的、被认可的，是人类对于事物的的妥协认知。“名”的全部不代表是“象”的全部；同样的，“征”的全部也不代表必然是“象”的全部。“名”的出现，体现主体对“象”的占有正在进行，这种占有需要“名”来固定。例如我看上了一宗地产，我若想占有它，就必须宣布这里属于我，而这是我对其附加的“名”。我们的身体行为，也是署名的工具，我无需语言只需要霸占一把长椅，就是在宣布长椅属于我。

为了固定“名”的认同，则会出现机体或精神的强制，督促认同这一“名”的存在，如果没有被承认的“名”，那么它将转瞬即去。

一切纷乱的根源在于，每一个人都妄图夺取任何事物的署名权，并认为他们的“名“就是“象”的全部。例如对一件事正义性的定义亦或是严肃性的定义，随着社会技术的演进，大量雇佣的网路水军与信息鼓动的出现，更让对一切事物的署名权争夺进入了激进的境地。

你可以通俗的认为，大量的虚假之“名”裹挟并吞噬真实的“象”，让人无法看到一切最真实自然的状态，而是一切遵循被设计的“名”状。人类在看到一切“象”（一切事物），逐渐只能看到他们的所有者（署名权所有人），他们的名“，包括被署名的价值、用途、归宿等等。真实性被剥夺，反之而来的是无数虚假之名的捆缚。这种捆缚包括了诞生了人类设计的既定秩序，例如财富的高低、地位的悬殊、又或是审美的取向。署名之争，让人类在人为中对事物形成了对立的取向，影响了人们对严肃、娱乐、正义，邪恶的取向。比如在一些地区堕胎是合法的，一些地区堕胎是违法的；一些地区谈论繁衍是公开娱乐化的，一些地区则是严肃隐私的。通过署名之争，能够更加纯粹的提取人群手中的署名权，以获得更加稳固的署名权力。

通过署名权的行使，人类和一切人能所见的事物被逐步划分，“象”被视作更多的“征”，“征”被界在为不胜其数的“名”，在争夺“名”的署名权下，引发了不可调和的纷争。
