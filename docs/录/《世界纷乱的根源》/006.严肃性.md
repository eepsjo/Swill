# 严肃性

> [原文链接](https://www.bilibili.com/video/BV1TfPDzSEYt/)

“名”，最简单的理解就是，主体对某一物（象）附加的认识。比如赋予某一土地作为祭祀用地、某一土地设为皇宫、某一土地设为平民居所。此时署名者为了维持个人的意愿与利益，为这些“名”，赋予了“严肃性”，比如禁止平民在皇宫里行走。

这片土地，本没有任何严肃性，所有建筑的材料、石头或木材，他们本身不具有严肃性。严肃性是人为赋予在“名”上的。严肃性的出现，成功将得到署名权的“大名主”获得了极大的自由与话语权。

从部落走到封建社会，每个人的署名权如同一瓢瓢水舀入一口大缸将署名权受让给了君主。这种出让原因很简单，即便将自己脚下世代耕种的的土地自己署名，但更强暴力征服为君主所有，在这种强制剥夺署名权的情形下，原本署名仍然失效。

君主可以随便支配署名权，在这种情形下，署名权的极度集中，就导致了君主认为象的名是什么，那么其他主体就如此看待存在。

著名权的强大与否的根源，就在于人口。拥有更多管辖人口的大名主（掌握最多署名权的人），就拥有更加强富裕的暑名权，归根结底是多少人能够向统治者出让自己的署名权。

“标准”是指对某一象（征）的界在的固定。比如我们称呼天上的火球为太阳。“标准”是“大名主”授意所界在的，大名主赋予了这些界在不可突破的严肃性。这些严肃性，封锁了人们对署名权的掌握。若有一个人朝着太阳说一句‘这是大火球＂，我想只能落得被嗤笑的下场，但他的这一做法，实际上是在重新夺回署名权。

太阳在大名主的著名下，已经成为严肃的界在，包括土地、法律、暴力机构，都被蒙上了严肃性的面纱，正如爱德华七世的肥胖难以扣上西服双排扣的第二粒扣子，使得日后只扣第一排扣子成为了固定的界在，并使之严肃起来，这是爱德华七世作为“大名主”的所作所为。而在无意识下，人类已经习惯出让对存在的署名权，主动解开了第二粒扣子。

署名权向大名主出让后，被冠以严肃性的头纱，在这样的情况下，界在就如此被固化，以至于你已无力收回署名权，因为大名主的署名被更多的人所认可。
