
| 阶段 | 内容 |
|------|------|
| Tokenizer | 逐字符扫描，识别数字/变量(q1-q4)/常量(pi,e,φ)/函数/运算符/括号 |
| Parser | 按标准表达式文法递归下降：`expr→term→factor→unary→primary`，`^` 右结合，`pow` 特化为二元函数 |
| Evaluator | 后序遍历 AST，变量代入滑块值，函数映射到 `Math.*`，除零/负数非整数次幂返回 NaN |
| 缓存 | 同一公式只解析一次，后续直接复用 AST |
