# 实用命令

- 更新 Windows Git
    ```
    git update-git-for-windows
    ```
- 分支重开
    ```
    git checkout --orphan latest_branch && \
    git add -A && \
    git commit -am "Initial commit" && \
    git branch -D main && \
    git branch -m main && \
    git push -f origin main
    ```
